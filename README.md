# React Villa Viewer

A modern React application that demonstrates image viewing functionality with a clean, interactive interface.

## Features

- 🏠 **Villa Image Viewer**: Click the "View Villa Image" button to display the villa image in a new tab
- ⚛️ **React + Vite**: Built with React and Vite for fast development and hot module replacement
- 🎨 **Modern UI**: Clean, responsive design with smooth transitions and hover effects
- 📱 **Responsive Design**: Works seamlessly on desktop and mobile devices

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/dmitry-ivanov47/react-villa-viewer.git
cd react-villa-viewer
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Usage

1. **View Villa Image**: Click the "View Villa Image" button to open the villa image in a new browser tab
2. **Counter**: Use the counter button to see React state management in action
3. **Explore Links**: Check out the documentation and social links in the footer
